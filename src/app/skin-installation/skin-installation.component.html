Installing

Automatic Steam Chat Skin Installer

The installer can apply the skin and provides basic customisation without needing to know where the skin files are located. This installer will only need to be run once.

Make sure to have friends.custom.css from the above customer in the same location as the installer

Manual

  • Download: friends.custom.css from above customiser or other sources
  • Move: friends.custom.css to Steam/clientui
