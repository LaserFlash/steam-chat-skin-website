Installing

New Steam update

Steam has release a large update, changing how we enable skins. You must now:

  • Have the latest version of SFP
  • Add skins to `Steam/steamui`

Windows / Linux / Mac OS

  • Download: friends.custom.css from above customiser or other sources
  • Move: friends.custom.css to Steam/steamui/<custom skin>
  • In SFP make sure the "Steam Skin" setting matches the folder you placed friends.custom.css in
  • You can apply this with other skins, just drop friends.custom.css into the other skin's folder
