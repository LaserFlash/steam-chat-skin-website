Installing

Automatic Steam Chat Skin Installer
Installer only works on Windows

The installer can apply the skin and provides basic customisation without needing to know where the skin files are located. This installer will only need to be run once.

Make sure to have friends.custom.css from the above customiser in the same location as the installer

Manual - Windows

  • Download: friends.custom.css from above customiser or other sources
  • Move: friends.custom.css to Steam/clientui

Manual - Linux & macOS

  • Download: webkit.css from above customiser or other sources
  • Move: webkit.css to Steam/skins/"skinName"/resource
  • You might want to combine webkit.css rather than replace to keep styles from your main client skin
